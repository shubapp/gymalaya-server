# Gymalaya!
An awesome mobile app built with phonegap for tracking fitness process.

## How to start it?
### Go to our website
1. [here](http://gymalaya.shubapp.com)

### Or run it on your own
1. `git clone https://github.com/shubapp/gymalaya-app.git`
2. `npm install`
3. `node server/server.js`
